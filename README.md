# console-stock

## 下载安装

```bash
wget https://raw.githubusercontent.com/u2takey/console-stock/master/main.php -O /usr/local/bin/stock
chmod +x /usr/local/bin/stock
```

## 使用

```bash
# search once, 支持 简写，代号等
stock cdkj 002185 00981

# watch，每隔 15 s 刷新一次
stock watch cdkj 002185 00981
```
